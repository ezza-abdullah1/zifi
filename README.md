# zifi

## Problem Statement

Measuring and monitoring internet speed is essential for diagnosing connectivity issues and ensuring optimal performance. Many users need a fast, reliable way to check their WiFi speed without installing complex software or dealing with unnecessary ads and clutter. This project delivers a simple web-based solution that allows users to check their WiFi download and upload speeds directly from their browser[attached_file:1].

## Technologies Used

- CSS for responsive styling and layout (50.8%)[attached_file:1]
- JavaScript for running speed test logic, capturing metrics, and displaying results (49.2%)[attached_file:1]

## Project Overview

`zifi` is a lightweight web application designed for quick and straightforward WiFi speed checks. It calculates and displays upload and download speeds, helping users assess the quality of their internet connection. The app is accessible at [zifi-iota.vercel.app](https://zifi-iota.vercel.app)[attached_file:1].

## Setup and Installation

1. **Clone the repository:**
    ```
    git clone https://github.com/ezza-abdullah1/zifi.git
    cd zifi
    ```

2. **Install dependencies:**
    - No extra dependencies are required. All application code uses basic HTML, CSS, and JavaScript.

3. **Run locally:**
    - Open `index.html` in your browser.
    - Or deploy using any static hosting service, such as Vercel, Netlify, or GitHub Pages.

## How to Use

- Open the application in your web browser.
- Click "Start Test" (or the equivalent button).
- The app will measure your WiFi speed by running several tests.
- Download and upload speeds will be displayed along with connection details.
